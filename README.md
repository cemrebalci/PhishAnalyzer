# PhishAnalyzer 🛡️
### AI-Powered Phishing URL Detection & Threat Analysis Platform

## Proje Hakkında
PhishAnalyzer, makine öğrenmesi kullanarak URL'lerin
phishing olup olmadığını tespit eden ve neden tehlikeli
olduğunu açıklayan bir web platformudur.

## Teknolojiler
- **Backend:** Python, Django REST Framework
- **Frontend:** React, Tailwind CSS
- **AI/ML:** Scikit-learn, Random Forest, XGBoost
- **Veritabanı:** PostgreSQL

## Model Performansı
- Dataset: 235.795 URL
- Algoritma karşılaştırması: Random Forest, XGBoost, Logistic Regression
- En iyi model: Random Forest — %99.99 doğruluk
